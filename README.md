# Ветеринарний Аудитор Дозування та Інфузій 🐾

Професійний медичний ветеринарний веб-застосунок на базі **Django**, розроблений для автоматизації складних розрахунків постійної інфузії (CRI), площі поверхні тіла (BSA) та аудиту сумісності критично важливих ліків з абсолютною математичною точністю.

---

## 🌟 Ключові можливості

1. **⚡ Калькулятор постійної інфузії (CRI)**:
   - Обчислює загальну концентрацію лікарського засобу у флаконі ($мг/мл$), швидкість інфузійного потоку ($мл/год$) та точну швидкість крапельниці ($крапель/хв$) на основі вибраного фактора системи.
   - Має вбудовану покрокову математичну верифікацію із формулами та прозорими розрахунками в реальному часі.

2. **📊 Токсикологічний калькулятор BSA**:
   - Розраховує площу поверхні тіла тварини ($м^2$) за допомогою формули Meeh і специфічних біологічних констант ($K$-факторів) для собак, котів, птахів та гризунів.
   - Обчислює абсолютне терапевтичне дозування на основі цільової дози на квадратний метр ($мг/м^2$).

3. **🧪 Інтерактивний аудитор сумісності ліків**:
   - Забезпечує миттєвий аудит суміші кількох лікарських засобів в одному шприці або інфузійній лінії для уникнення загроз фізичного та хімічного осаду чи інактивації препаратів.
   - Включає повну інтерактивну клінічну матрицю сумісності для 5 життєво важливих засобів (*Пропофол, Діазепам, Кальцію глюконат, Мідазолам, Фуросемід*).

---

## 📐 Клінічна точність та безпека

- **Високоточна математика**: Для розрахунків використовується бібліотека Python `decimal` з режимом округлення `ROUND_HALF_UP`, що повністю усуває похибки двійкового представлення чисел із рухомою комою.
- **Масштаб Meeh-формули**: Розрахунок площі поверхні тіла використовує точне масштабування маси в грамах із поділом на $10,000$, що гарантує фізіологічно коректні результати (наприклад, $BSA = 0.47\text{ }м^2$ для собаки масою $10\text{ кг}$).

## 🔧 Встановлення та запуск

1. **Встановіть залежності**:
   Переконайтеся, що на комп'ютері встановлено Python 3.10+, відкрийте термінал у папці проекту та виконайте:
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустіть вбудовані міграції**:
   ```bash
   python manage.py migrate
   ```

3. **Запустіть локальний сервер розробки**:
   ```bash
   python manage.py runserver
   ```

4. **Відкрийте застосунок**:
   Перейдіть у браузері за адресою [http://127.0.0.1:8000/](http://127.0.0.1:8000/).
