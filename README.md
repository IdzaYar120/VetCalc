# Ветеринарний Аудитор Дозування та Інфузій 🐾

Професійний медичний ветеринарний веб-застосунок на базі **Django**, розроблений для автоматизації складних розрахунків постійної інфузії (CRI), площі поверхні тіла (BSA), гемотрансфузії, наркозу та аудиту сумісності критично важливих ліків з абсолютною математичною точністю.

---

## 🌟 Ключові можливості

1. **⚡ Калькулятор постійної інфузії (CRI) та інфузійної терапії**:
   - Розрахунок CRI, обсягу інфузії (підтримуюча + дефіцит + втрати), швидкості крапельниці (фактор системи) з урахуванням виду тварини.
   - Спеціалізований калькулятор додавання калію в інфузію з автоматичним аудитом кардіологічного ліміту (K-max).
   - **Гемотрансфузія**: точний розрахунок необхідного об'єму донорської крові для покриття дефіциту гематокриту пацієнта (для котів та собак).

2. **🏥 Медичне розширення (Реанімація, Електроліти, Наркоз)**:
   - **Emergency/CPR**: розрахунок доз ургентних препаратів (Адреналін, Атропін, Лідокаїн, Налоксон, Дексаметазон) з детальними протоколами безпеки (за стандартами RECOVER Initiative).
   - **Розрахунок наркозу**: автоматизований підбір дозувань для премедикації та індукції наркозу (Пропофол, Альфаксалон, Кетамін, Дексмедетомідин, Буторфанол).
   - **Коригований кальцій**: автоматична корекція рівня загального кальцію за рівнем альбуміну.
   - **Дефіцит бікарбонату (NaHCO₃)**: розрахунок дефіциту основ (Base Deficit) для корекції метаболічного ацидозу.
   - **Осмолярність плазми**: визначення плазмової осмолярності на основі натрію, глюкози та сечовини.

3. **📊 Токсикологічний калькулятор BSA**:
   - Розраховує площу поверхні тіла тварини ($м^2$) за допомогою формули Meeh і специфічних біологічних констант ($K$-факторів) для собак, котів, птахів та гризунів.

4. **🧪 Інтерактивний аудитор сумісності ліків**:
   - Забезпечує миттєвий аудит суміші кількох лікарських засобів в одному шприці або інфузійній лінії для уникнення загроз фізичного та хімічного осаду.

5. **📇 Локальний архів пацієнтів та Друк (IndexedDB)**:
   - **Офлайн архів**: збереження розрахунків, діагнозів та параметрів пацієнтів безпосередньо у внутрішню базу даних браузера (IndexedDB). Дані зберігаються навіть без інтернету!
   - **PDF Експорт**: миттєва генерація та друк клінічних «Листів призначення» для будь-якого калькулятора зі збереженням історії аудиту.
   - **Відновлення сесій**: завантаження збережених параметрів пацієнта з архіву назад у калькулятор одним кліком.

6. **📱 Сучасний PWA Інтерфейс (Light/Dark Mode)**:
   - Преміальний дизайн із підтримкою скляних елементів (Glassmorphism), іконок Lucide та мікроанімацій.
   - **Offline Mode**: інтеграція Service Worker для повноцінної роботи додатка без доступу до інтернету.

## 📐 Клінічна точність та Архітектура

- **Транспіляція Python у JavaScript (`core/transpiler.py`)**: Вирішення проблеми "Source of Truth" (єдиного джерела істини). Спеціальний механізм автоматично транспілює еталонні Python-функції розрахунків у клієнтський JS код під час запуску сервера. Це гарантує абсолютну консистентність математики між бекендом (Python) та фронтендом (JS), дозволяючи калькулятору безпечно працювати в офлайн-режимі.
- **Високоточна математика**: Для розрахунків використовуються спеціалізовані підходи, що повністю усувають похибки двійкового представлення чисел із рухомою комою.
- **Масштаб Meeh-формули**: Розрахунок площі поверхні тіла використовує точне масштабування маси в грамах із поділом на $10,000$, що гарантує фізіологічно коректні результати.

## 🔧 Встановлення та запуск

1. **Встановіть залежності**:
   Переконайтеся, що на комп'ютері встановлено Python 3.10+, відкрийте термінал у папці проекту та виконайте:
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустіть вбудовані міграції**:
   ```bash
   python manage.py migrate
   ```

3. **Запустіть локальний сервер розробки**:
   При запуску сервера автоматично генерується офлайн JS-файл калькуляторів (`calculators_offline.js`).
   ```bash
   python manage.py runserver
   ```

4. **Відкрийте застосунок**:
   Перейдіть у браузері за адресою [http://127.0.0.1:8000/](http://127.0.0.1:8000/).
